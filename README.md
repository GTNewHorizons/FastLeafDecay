FastLeafDecay
=============
A simple Minecraft mod that speeds up leaf decay after you cut down the wood in a tree.

## Requirements
* Minecraft 1.7.10
* [Minecraft Forge](http://files.minecraftforge.net/) version 10.13.0.1199 or higher
